Busca endereço por CEP

  1. Criar um formulário para endereço, com os campos:
    • CEP - obrigatórios
    • Rua - obrigatórios
    • Número - obrigatórios
    • Complemento - opcional
    • Bairro - obrigatórios
    • Cidade - obrigatórios
  2. Criar uma função JS que recebe um CEP e busca os dados na api da viacep
  3. Conectar essa função ao evento de onChange do campo de CEP e preencer os demais campos com os valores
  4. Tornar os demais campos readonly durante a busca
  5. Voltar os campos para editáveis caso a busca por cep dê algum erro
